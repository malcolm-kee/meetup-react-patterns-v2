## Higher Order Component (React's decorator)

```js
const EnhancedComponent = higherOrderComponent(WrappedComponent);
```

```js
// definition
const enhance = (Component) => {

    const EnhancedComponent = doSomething(Component);

    return EnhancedComponent;
}

// usage
const EnhancedList = enhance(List);
```

## Compound Component 

```jsx
<select>
    <option value="1">One</option>
    <option value="2">Two</option>
    <option value="3">Three</option>
</select>
```

---

|Patterns           |Description                            |Used when                              |
|-------------------|---------------------------------------|---------------------------------------|
|Render Props       |Delegates UI to component user         |Reuse functionality                    |
|HOC                |Injects props to wrapped component     |Same as above, with different API      |
|Compound Component |Parent injects props to child component|Multiple components that works together|
